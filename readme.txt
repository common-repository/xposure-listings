== Description ==

The Xposure Listings plugin will allow you to add the list of your current, up to date Xposure listings to your website.
The listings are available as a Gutenburg Block, as well as a shortcode.
You must have an Xposure account in order to embed your listings.

To get started using a Block:

* Add the Xposure Listings block to your page
* View the block settings screen, and enter your Xposure Agent ID in the appropriate text box.

Using a shortcode:

* use the shortcode [xplistings agt_id="xxxx"] where xxxx is your Xposure Agent ID

If you don't know your Xposure Agent ID, please log into the Xposure Control Center, and look in the Worpress Listings Plugin section under the Web tab
